newT.js

Tasty JavaScript Templating.

You love building JavaScript apps. Armed with the latest tools, you're dying for a tasty little JS template, that has a friendly syntax, low learning curve and lots of room to grow. Enter newT.js, tasty JavaScript templating.

With a node to HAML, and much love to super templates like mustache.js, newT.js is a clientside templating engine with the power and flexibility you need for building powerful, response, Asynchronous JavaScript applications. newT uses a syntax that elimantes the tedious closing and balancing of HTML tags, while still providing safe templates to defend against persistent XSS attacks common in dynamic JS applications in the HTML5 world.

Add newT.js

<script src="newT.js" type="text/javascript"></script>
